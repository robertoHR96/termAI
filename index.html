Modo Comando Directo

user@host:~$ termai "crea un servidor web simple en python"

🚀 Comando a ejecutar:

python -m http.server 8000

📝 Explicación:

Inicia un servidor web HTTP básico en el puerto 8000, sirviendo los archivos del directorio actual.

Instalación y Descarga

1. Clonar el repositorio (Recomendado)

git clone https://github.com/robertoHR96/termAI.git
cd termAI

O descarga el script directamente:

curl -O https://raw.githubusercontent.com/robertoHR96/termAI/main/termAI.py

2. Configurar Entorno Virtual e Instalar Dependencias

Es crucial usar un entorno virtual para evitar conflictos de paquetes:

python3 -m venv .venv
./.venv/bin/pip install llama-cpp-python

3. Hacer el Script Ejecutable y Crear Acceso Directo

Para usar `termai` como un comando del sistema:

chmod +x termAI.py
sudo ln -s $(pwd)/termAI.py /usr/local/bin/termai

*(Necesitarás tu contraseña para `sudo`)*

4. Iniciar el Servidor (Opcional, se inicia automáticamente)

El servidor se iniciará automáticamente la primera vez que uses `termai`. Si deseas iniciarlo manualmente en segundo plano:

termai --start-server

5. Usar termAI

Ahora puedes usar `termai` de dos maneras:

Modo Comando:

termai "crea un servidor web simple en python"

Modo Chat Interactivo:

termai

*(Escribe 'salir', 'exit' o 'quit' para terminar el chat)*

6. Detener el Servidor (Cuando no lo necesites)

Para detener el proceso del servidor en segundo plano:

termai --stop-server
